# MCP Guide

You have access to a rich ecosystem of MCPs (Model Context Protocol). Check `AGENTS.md` and `.mcp.json` for the full list.

## Quick Selection Guide

- **Web Search & Research**: Use `tavily`, `exa-search`, `brave-search`, `fetch`, `arxiv`.
- **Food & Life**: Use `swiggy-food`, `swiggy-instamart`, `swiggy-dineout`, `zomato-mcp`, `weather`, `google-calendar`.
- **Development**: Use `filesystem`, `github`, `context7`, `excalidraw`.
- **Data & Ops**: Use `postgres`, `sqlite`, `redis`, `prometheus`.
- **Agent Logic**: Use `sequential-thinking` for complex problems, `memory` for persistence.
